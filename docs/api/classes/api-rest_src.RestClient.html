    Class RestClient

    HTTP Client for REST requests. Send and receive JSON data. Sign request with AWS credentials if available Usage:

    const restClient = new RestClient();
restClient.get('...')
    .then(function(data) {
        console.log(data);
    })
    .catch(err => console.log(err));
